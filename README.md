# ElrsDual

**ElrsDual** is a specialized fork of ExpressLRS (on the `elrsdual` origin) designed to provide advanced functionality for dual-boot and multi-slot management systems.

## Features
ElrsDual introduces specific enhancements over standard ExpressLRS, focusing on:
- **Dual-OTA Capabilities**: Enhanced management of firmware updates across multiple slots.
- **Bootloader Slot Switching**: Seamless logic for managing different boot states.
- **Robust Configuration**: Optimized specifically for the requirements of the ElrsDual ecosystem.
