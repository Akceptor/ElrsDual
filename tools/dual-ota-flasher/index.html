ELRS Dual-OTA Flasher

Flashes two firmware versions into the OTA slots of one ESP32 (app0 = ELRS v3.x, app1 = ELRS v4.x) and reads them back. Chrome/Edge only.

Web Serial is not available in this browser. Use Chrome or Edge over http://localhost or https.

Build from source (GitHub Actions)

Compiles a chosen ESP32 target from the v3.6.3 / v4 branch in GitHub Actions, configures it here in your browser (the bind phrase never leaves this page), and stages it into an OTA slot. Then Connect and use Flash staged → app0/app1.

Slot-switch bootloader

Installs the custom second-stage bootloader (to 0x1000 only; app slots untouched). With it, 3 quick power cycles (each off/on within ~2 s) flips the active slot — no computer needed. Normal boots wait ~2 s before starting. Built for 4 MB ESP32 / min_spiffs.
